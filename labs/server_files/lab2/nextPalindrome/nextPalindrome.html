Write a function nextPalindrome(n) which returns the smallest integer m larger than n such that m is a palindrome. 
For example, nextPalindrome(9) should return 11, and nextPalindrome(12) should return 22.

CONSTRAINTS
n ≤ 105 for test cases worth a total of 70 points.
n ≤ 1021 for final test case worth 30 points.

TIME LIMIT
150ms

INPUT FORMAT
The first line of the input contains n.

OUTPUT FORMAT
Please output the smallest integer m larger than n such that m is a palindrome.
