Exercise 1 
Consider the Trionacci sequence defined as follows. T(i) = 1 if i=0 or i=1 or i=2, 
and T(i) = T(i-1) + T(i-2) + T(i-3) otherwise. Implement a function trionacci(N)
which returns the N-th Trionacci number.

CONSTRAINTS
0 ≤ N ≤ 9 for 3 test cases worth a total of 30 points.
0 ≤ N ≤ 20 for 2 test cases worth a total of 40 points.
0 ≤ N ≤ 25 for the final test case worth 30 points.

TIME LIMIT
150 milliseconds for the first 5 test cases.
300 milliseconds for the final test case.

INPUT FORMAT
A single line containing the integer, N.

OUTPUT FORMAT
A single line containing the Nth trionacci number.
