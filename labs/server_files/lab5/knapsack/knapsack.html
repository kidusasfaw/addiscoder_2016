Exercise 5 
Given a knapsack with a given volume and a list of items with given volume and value, 
return the maximum value of a knapsack without exceeding the knapsack volume.

CONSTRAINTS
For all items in all test cases: 
1 ≤ volume, value ≤ 100

For the volume of the knapsack:
1 ≤ volume ≤ 1,000

TIME LIMIT
1 second per test case.

INPUT FORMAT
The first line gives the volume of the knapsack.
Each other line represents a single item in the list of items of the form volume value.

OUTPUT FORMAT
A single line containing the maximum value of a knapsack.

SAMPLE INPUT
10
7 10
5 6
4 5

SAMPLE OUTPUT
11
