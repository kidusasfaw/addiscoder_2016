Exercise 3 
Given a list L of integers, return the longest strictly increasing subsequence of L.

CONSTRAINTS
For all test cases: 
For A in L:
1 ≤ A ≤ 1,000

TIME LIMIT
1 second per test case.

INPUT FORMAT
A single line containing the list of integers.

OUTPUT FORMAT
A single line containing a subsequence list of integers

SAMPLE INPUT
1 3 5 2 7 9

SAMPLE OUTPUT
1 3 5 7 9
