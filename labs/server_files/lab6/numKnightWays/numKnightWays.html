A superknight is on a chessboard, initially at grid location (0,0) (the bottom left corner).

Write a function numKnightWays(x,y,L) that returns n, the number of ways for a superknight
to get to location (x,y) if his allowed moves are given in the list L.

Each element in L is a list of size two signifying that it is possible for the knight to move
from (a,b) to (a+i,b+j). i and j are always both positive.

CONSTRAINTS
|L| ≤ 10 for all test cases
i,j ≤ 20 (for all pairs i,j in L) for test cases worth a total of 70 points
i,j ≤ 100 (for all pairs i,j in L) for final test case worth 30 points
x,y ≤ 20 for test cases worth a total of 70 points
x,y ≤ 100 for final test case worth 30 points

TIME LIMIT
200ms

INPUT FORMAT
The first line of the input contains three space separated integers, x, y, and |L| (the
length of the list L).

Each of the remaining |L| lines contain two space separated integers, ik and jk,
indicating that it is possible for the superknight to move ik units horizontally and
jk units vertically in a single move.

OUTPUT FORMAT
Please output the number of ways for the superknight to get to (x,y), beginning at (0,0).
