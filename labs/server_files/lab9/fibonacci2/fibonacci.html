The fibonacci sequence F is defined by the recurrence relation F0 = 1, F1 = 1,
and Fn = Fn-1 + Fn-2 for all n ≥ 2.

Write a function fibonacci(n) for computing the nth Fibonacci number taken
mod 1000000007 (109+7), using repeated squaring.  

CONSTRAINTS
n ≤ 20 for first the four test cases worth a total of 30 points
n ≤ 106 for the single test case worth 30 points
n ≤ 1012 for the single test case worth 40 points

TIME LIMIT
200ms

INPUT FORMAT
The first line of the input contains a single integer n.

OUTPUT FORMAT
Please output the value of the nth term of the Fibonacci sequence mod 1000000007.
