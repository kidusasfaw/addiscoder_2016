Exercise 1 
Implement either selectionSort, insertionSort, bubbleSort, or mergeSort to sort an array of N integers.

CONSTRAINTS:
1 < N < 1000

TIME LIMIT:
1 second

INPUT FORMAT:
A single line containing the integer N, followed by N lines each containing 1 of the N integers in the array to be sorted.

OUTPUT FORMAT:
A single line containing the N integers, space separated, in (non-strictly) increasing order.

SAMPLE INPUT:
4
1
5
3
6

SAMPLE OUTPUT:
1 3 5 6
