Problem B 
Recall the Trionacci sequence defined in lab 3:
	|1				if i=0 or i=1 or i=2
T (i) =	|
	|T (i-1) + T (i-2) + T (i-3)	otherwise

Implement a function trionacci(n) which returns the nth Trionacci number.
Your function should only require O(log2n) integer multiplications.

CONSTRAINTS
n ≤ 10000.

INPUT FORMAT
The first line of the input is T, the number of test cases. Then, for each of the test cases,
there is one line with the value for n.

OUTPUT FORMAT
For each test case, one line with the nth trionacci number.
