Lab 10: Exercise 1 part 2 
Write a function fromMatrixToList() which takes an adjacency matrix 
representation of a graph and outputs an adjacency list representation of the same graph. 

TIME LIMIT
1 second per test case. 

INPUT FORMAT
A number representing the number of nodes in the graph. Each following line is a line of
the adjacency matrix.

OUTPUT FORMAT
A nested list representing the adjacency list form of the graph.

SAMPLE INPUT
2
0 1
0 0

SAMPLE OUTPUT
[[1],[]]
