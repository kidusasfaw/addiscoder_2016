Lab 10: Exercise 4 
Write a functionnumComponents(A) which takes an adjacency list A and outputsthe number of 
connected components in the graph represented by A 

TIME LIMIT
1 second per test case. 

INPUT FORMAT
The input will be a number n which represents the number of nodes in the graph. Following n
will be a sorted list of nodes for which an edge from node zero to that node. The next line
will be a similar list for node 1 and so on until node n.



OUTPUT FORMAT
An int representing the number of connected components in the graph

SAMPLE INPUT
4
1
2
3
0


SAMPLE OUTPUT
1
