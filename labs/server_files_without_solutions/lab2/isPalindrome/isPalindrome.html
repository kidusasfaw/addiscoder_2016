An integer is said to be a palindrome if its digits are
the same forward and backwards(not including leading
zeroes). For example, 12321 is a palindrome, as is 5. 1231
on the other hand is not a palindrome, and neither is 50
(remember we are not including leading zeroes). Write a 
function isPalindrome(n) which returns True if n is 
a palindrome and False otherwise.

CONSTRAINTS
1 ≤ n ≤ 10e16

TIME LIMIT
100 ms

INPUT FORMAT
A single integer n

OUTPUT FORMAT
A boolean value, True or False
