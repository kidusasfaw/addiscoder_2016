Exercise 2 

Given an integer n and a list L of distinct integers, find whether or not there exist two distinct integers in the list that sum to n.

CONSTRAINTS:
1 < n < 10000
1 < len(L) < 500

INPUT:
An integer n, followed by an integer len(L), followed by len(L) distinct integers representing the list L.

OUTPUT:
Two integers in increasing order summing to n, or False if no such integers exist.

SAMPLE INPUT:
4 3 1 3 4

SAMPLE OUTPUT:
1 3
