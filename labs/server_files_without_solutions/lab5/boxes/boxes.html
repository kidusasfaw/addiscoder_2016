Given a list of boxes with height, width, and depth, return the tallest stack of boxes
such that each box has strictly smaller length and width of the base than the box directly below it. Note that boxes can be rotated to satisfy this constraint.

CONSTRAINTS
For all boxes in all test cases: 
1 ≤ width, height, depth ≤ 40
Maximum of 50 boxes in the test cases.

TIME LIMIT
≤ 1 second per test case.

INPUT FORMAT
Multiple line containing the list of boxes where each line is each box's width, height, and depth.

OUTPUT FORMAT
A single line containing the maximum height of a box.

SAMPLE INPUT
6 7 2
2 4 1
1 1 5

SAMPLE OUTPUT
8
