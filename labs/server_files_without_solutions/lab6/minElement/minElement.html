Problem C 
Write a recursive procedure minElement(L) which takes a list L of integers and
returns the minimum element in the list.

CONSTRAINTS
The length of the list must be between 0 and 20 inclusive.

TIME LIMIT
1 second per test case

INPUT FORMAT
The first line of the input is T, the number of test cases. Then, for each of the test cases,
there is one line with the list elements such that L[0] appears first, L[1] appears second, etc.

OUTPUT FORMAT
For each test case, print out one line with the minimum element in the list.
