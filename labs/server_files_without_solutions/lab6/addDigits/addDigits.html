Exercise 1 
Given a nonnegative integer n, return the sum of the digits of n. 

CONSTRAINTS
For all items in all test cases: 
1 ≤ n ≤ 10e15

TIME LIMIT
1 second per test case.

INPUT FORMAT
A single line giving integer n.

OUTPUT FORMAT
A single line containing the sum of the digits of n.

SAMPLE INPUT
12345

SAMPLE OUTPUT
15
